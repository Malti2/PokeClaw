Installation

Set up PokeClaw in minutes.

Install Clone the repo and switch to beta. 
git clone https://github.com/Malti2/PokeClaw.git
cd PokeClaw
git checkout beta
Use Launch the app and connect your workflow. 
# Start the server and open the Mac app
npm run dev
open -a PokeClaw
